SakyContextMenu

설명
윈도우키를 이용해 수신호, 조명부착, 깃발부착, 무기홀스터, 귀마개설정을 수행할 수 있는 편의 스크립트입니다.

설치 방법
1. 미션파일 최상위 경로에 SakyContextMenu 폴더를 추가합니다.
2. description.ext 파일 내에 아래의 코드를 추가합니다.

#include "SakyContextMenu\dialog.hpp"
class cfgFunctions {
	#include "SakyContextMenu\functions.hpp"
};

cfgFunctions 클래스가 이미 선언되어 있는 경우에는 해당 클래스 내부에 
#include "SakyContextMenu\functions.hpp" 
구문만 추가합니다.
